# 一个关于阅读的 Android 的简单应用

## 功能特点

- 有封面，有目录，分章节阅读
- 背景和文字搭配好，阅读舒畅
- 书签功能及下次进入程序回到正在观看的文字位置功能
- 能够改变字体大小，颜色和背景色

## 内容简介 
精选 226 篇鬼故事，包含校园鬼故事，医院鬼故事，家里鬼故事，民间鬼故事，以短篇为主，故事引人入胜，篇篇精彩，无尿点。

## 简短介绍
精彩的鬼故事精选，你晚上敢看吗？

v3.0
